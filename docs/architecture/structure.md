# Frontend Structure

Project follows feature/layer separation:

- `src/pages/` route-level composition.
- `src/components/` reusable UI blocks.
- `src/hooks/` stateful logic and side effects.
- `src/services/` API and logger utilities.
- `src/types/` request/response contracts and parsers.
- `src/routes/` centralized route config.
